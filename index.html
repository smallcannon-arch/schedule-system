⏱️ 教師節數檢核

      
實配 vs（角色基準＋超鐘點−減課）；含導師課

      
    
    
    
  
    

    



    
  
    🌐 語言分組鎖定
    
  
    選用功能。用於客語、原住民族語、直播共學及其他抽離分組；原班閩南語請至「資料建置 ＞ 配課」設定。
    
  
    
    
  
    
  
    
    
    
      操作說明
      
      
        
    1. 設定共同時段：每個年級只設定一次星期與節次，系統會鎖定該年級所有班級課表。
      2. 
        
    2. 建立抽離分組：選擇語別、來源班級與授課教師；年級會由來源班級自動判斷。
      3. 
        
    3. 重新排課：任何異動都會使舊課表失效，請重新執行 CP-SAT 並確認硬規則為零違反。
      4. 
      
    儲存後會鎖定全年級班級課表、授課與協同教師，以及非原班教室；導師與課表編修均不能移動。
    
    
    
    
    
      
    班級共同時段
    
      
    每個年級設定一個固定節次；班級課表只顯示「本土語文（分組上課）」，不重複計算各分組教師。
    
      
    
      
    
    
    
    
    
      
    語言抽離分組
    
      
    主畫面只保留語別、來源班級、授課教師與教室；人數、協同教師、方式及自訂組名需要時再展開。
    
      
      
    
      
    
        
      
    
      
    此處只建立抽離或直播分組；閩南語原班教師與班級請在「資料建置 ＞ 配課」維護。
    
    
    
  
    

    



    
  
    🧩 資源班
    
  
    一個抽離分組可同時包含同年級多個原班，資源班教師節數只計一次。資源班授課科目與原班可抽離科目分開設定，也可由教師先鎖定實際上課時段。
    
  
    版本 1.26 相容性提醒：既有案件若曾以「閩南語」建立固定課，載入後可能恢復先前未生效的鎖定；請重新執行排課並檢查無解診斷後再發布。
    
  
    
    操作說明
    
      
      
    1. 建立分組：同一組可勾選同年級多個來源班級，例如一年甲、乙班合併為 A 組。
      2. 
      
    2. 分開選科：「資源班授課科目」是教師實際教授內容；「原班可抽離科目」可複選國語、數學、彈性或綜合等課程。
      3. 
      
    3. 安排時段：可交由系統安排，也可切換固定時段後直接點選每週節次。
      4. 
      
    4. 早自修：只供資源班固定時段使用，會顯示在班級與教師課表，但不會寫入人力資源網或校務系統上傳檔。
      5. 
    
    資源班綁課屬於鎖定課程，導師與一般課表編修都不能移動。
    
  
    
  
    
    
    已納入抽離的原班
    
    
    
  
    
  
    
    
    抽離分組
    
    
    
    
    
    
    
    
    系統會讓所有來源班級在同一節抽離，並自動避開資源班教師衝堂；同一分組不會因來源班級數量重複計算教師節數。
    
  
    

    



    
  
    🚫 不排課時間
    
  
    對象可填教師姓名（該師不排）、年級如「3年級」（學年共同時間）、班級如「3甲」。修改後重新排課即生效。
    
  
    
    操作說明
    
      
      
    1. 年級共用：學年共同時間可直接選年級後點選時段，快速套用到全年級。
      2. 
      
    2. 教師時段：鐘點、兼課或跨校教師可使用教師快速設定，直接點選整週可排與不可排時段。
      3. 
      
    3. 個別限制：單一班級、特殊原因或例外規則，使用限制清單逐列新增。
      4. 
      
    4. 重新求解：任何限制異動後都要重新執行排課，舊結果不會自動重算。
      5. 
    
    限制過多可能造成無解；若求解失敗，先檢查同一教師或班級是否被封鎖過多時段。
    
  
    
  
    
    
    ⚡ 年級快速設定（學年共同時間）
    
    
    選年級後直接點格子切換：粉紅＝該年級此節不排；灰＝本來就不可排。一次點好整個學年，免逐班設定。
    
    
    
      
      
    
    
    
  
    
  
    
    
    👤 教師快速設定
    
    
    適合鐘點、兼課與跨校教師。先選教師，再直接點選每週時段；「看可排時段」搭配「先全部設為不可排」，可以快速留下實際到校的時段。

    📐 排課規則
    
  
    硬規則＝絕不違反。瀏覽器 DEMO 會驗證衝堂、場地、時段、連堂、鎖定與每日上限；軟規則權重僅由正式版 CP-SAT 引擎最佳化，此頁不再假裝已套用權重。
    
  
    
    操作說明
    
      
      
    1. 先看硬規則：衝堂、鎖定、場地與不可排時段等條件必須全部成立。
      2. 
      
    2. 再調軟規則：上午優先、分散或集中等偏好，由 CP-SAT 依權重盡量最佳化。
      3. 
      
    3. 少量新增：只有學校確實需要的條件才新增自訂規則，避免互相牴觸。
      4. 
    
    硬規則互相衝突會直接無解；軟規則未完全滿足不代表課表無效。
    
  
    
  
    
    
    
    
    
  
    

    



    
  
    ⚡ 執行排課
    
  
    瀏覽器 DEMO 適合快速預覽；正式成果請使用 CP-SAT API，並以完整度與硬規則檢核結果為準。
    
  
    
    操作說明
    
      
      
    1. 排課前：先確認配課完整，資源班、不排課時間與規則都已設定。
      2. 
      
    2. 開始求解：CP-SAT 先排科任、固定課與鎖定課；導師課預設保留給老師登入後自行安排。
      3. 
      
    3. 檢查結果：確認排入節數、未排項目與硬規則違反，再進行課表編修。
      4. 
    
    正式使用應以硬規則違反為 0 且所有必要節數完整排入為準。
