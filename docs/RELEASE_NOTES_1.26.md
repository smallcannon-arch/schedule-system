# 排課輔助系統 1.26

## 資源班流程

- 修正新增抽離組後只能選擇一年級班級的問題；切換年級時會一次替換來源班級，避免形成混年級分組。
- 「資源班學生」改為可保存的班級資格標記，不再因分組正規化而被自動清除。
- 資源班教師選單優先顯示「資源班教師」與「資源班導師」，並保留其他兼任教師選項。
- 固定時段會依來源年級的實際上課時段禁用不可用節次，送出前也會顯示具體錯誤。
- 新增資源班專用「早自修」固定時段；不占原班正式課程節數，但會檢查教師與來源班級重複衝突。班級與教師課表會顯示，人力資源網與校務系統上傳檔預設排除。

## 科目相容性

- 科目解析改為「學校原名優先，標準別名其次」。學校同時建立「閩南語」與「本土語文」時，不會把閩南語錯配到本土語文。
- 固定課使用同一解析規則。舊案件中先前因別名轉換而被忽略的「閩南語」固定課，載入後可能恢復生效。

## 既有學校注意事項

載入既有雲端案件後，請重新執行一次排課並檢查無解診斷、固定課與資源班時段，再發布正式教師課表。若原案件曾同時建立「閩南語」與「本土語文」，請特別核對固定課位置。
