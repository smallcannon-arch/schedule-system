# 排課輔助系統 1.28

## 資源班排課流程

- 新增抽離分組時，預設先指定資源班每週上課時段；尚未排定時仍可改由系統尋找。
- 一般節次預設以國語文、數學作為原班可抽離科目，其他科目收合為選用設定。
- 同一分組、同一節次的所有來源班級，必須排入相同的原班可抽離科目。
- 早自修維持資源班獨立固定時段，不綁定原班科目。

## 相容性

- 既有案件已選擇的原班可抽離科目與安排方式不會被覆寫。
- 舊格式且尚未設定抽離科目的分組，載入時會以國語文、數學作為預設。
- 既有資源班分組若曾讓來源班級在同一節抽離不同科目，重新排課時會改判為無解；請調整原班固定課，使各班同節同科後再執行。
